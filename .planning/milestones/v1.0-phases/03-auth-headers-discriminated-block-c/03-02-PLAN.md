## Goal

Replace ONE call in Forge's `X9Client` class (the `reload(agentId)` method) with
`createBridgeClient` from the bridge, proving the typed HTTP client works in a real
consumer. After this migration, calling `reload()` without a properly typed
`X-Internal-Secret` header is a compile-time error. This is the pilot migration that
validates the Phase 3 auth pattern before Phase 4 migrates all 11 endpoints.

Success Criterion #4 from Phase 3: "Forge `X9Client` aggiornato a usare `createBridgeClient`
per almeno `/internal/agents/:id/reload` (pilota)".

<action>
Modify `x9.client.ts` to:

1. Add imports at top of file:
```typescript
import { createBridgeClient, type BridgeClient } from '@x9-forge/contracts/http';
```

2. Add a private `bridgeClient` field initialized in the constructor:
```typescript
private readonly bridgeClient: BridgeClient<'secret'> | null;
```

In the constructor, after existing init:
```typescript
this.bridgeClient = internalSecret
  ? createBridgeClient({ baseUrl, auth: { 'X-Internal-Secret': internalSecret } })
  : null;
```

3. Replace the `reload(agentId: string)` method body. Current implementation likely does:
```typescript
async reload(agentId: string): Promise<void> {
  await fetch(`${this.baseUrl}/internal/agents/${agentId}/reload`, {
    method: 'POST',
    headers: this.internalHeaders(),
  });
}
```

Replace with:
```typescript
async reload(agentId: string): Promise<void> {
  if (!this.bridgeClient) {
    throw new Error('Cannot reload agent: no internal secret configured');
  }
  await this.bridgeClient.request({
    method: 'POST',
    path: `/internal/agents/${agentId}/reload`,
  });
}
```

4. Keep ALL other methods (`stop`, `listAgents`, `discoverCapabilities`, `getEnvSchema`)
   using the existing `this.internalHeaders()` pattern — they migrate in Phase 4.

5. Keep the existing `internalHeaders()` helper — still used by non-migrated methods.

Key decisions:
- `bridgeClient` is `null` when no `internalSecret` provided (Docker trust fallback)
- `reload()` throws explicitly instead of sending empty header silently (Bug #15 fix class)
- Only `reload()` migrated — minimal blast radius for pilot
- `BridgeClient<'secret'>` type ensures only `X-Internal-Secret` endpoints are callable
</action>
